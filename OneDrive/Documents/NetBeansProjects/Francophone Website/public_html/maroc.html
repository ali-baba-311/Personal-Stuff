Bienvenue au Maroc!

Musicien Honoré: French Montana

  • Son nom de scène est French Montana, mais son nom
    de naissance est Karim Kharbouch
  • Il est né dans le Maroc, mais il habitait dans les États-Unis pour la majorité de sa vie
  • Il est très célèbre pour ces chansons qui sont très vite et dynamique
